# n8n Factory Frontend - 實作總覽

我已成功開發並優化了 **n8n 自動化整合工廠** 的前端顯示介面。現在它不僅具備「Premium」的視覺質感，且完全適配您的本地開發環境。

## 系統架構升級

在您的架構 `admin <-> frontend <-> backend <-> n8n` 中，前端現在扮演著強大的「數據中控台」角色：
- **視覺風格**：導入 **Tailwind CSS v4** 與 **Glassmorphism (毛玻璃)** 設計，搭配 `Outfit` 字體與動態漸層背景。
- **即時監控**：整合 WebSocket，實現「即時偵測日誌 (Live Trace)」，不需重新整理即可看到 LINE Bot 的一舉一動。
- **動態適配**：自動偵測後端 IP（預設指向 `.147`），確保在區網內切換設備也能正常連線。

## 核心功能實作

### 1. 繁體中文在地化 (Localization)
- **控制儀表板**：顯示線索總數、進行中任務與 Bot 指令執行量。
- **線索管理**：優化表格顯示，支援頭像（訪客/用戶）、狀態標籤與 UID 顯示。
- **數據分析**：加入互動趨勢圖表與指標監測。

### 2. 多層次動畫感 (UX Enhancements)
- 使用 **Framer Motion** 實作元件進場動畫。
- 切換分頁時擁有平滑的過渡效果。
- 系統日誌加入自動滾動與強調動畫。

### 3. 工程架構優化
- **Vite 8 + Tailwind v4**：使用最尖端的開發工具鏈。
- **連線重連機制**：WebSocket 斷線時會自動嘗試重新連結，保持監控穩定。

## 如何運行

1. **進入目錄**：`cd frontend`
2. **啟動開發伺服器**：`npm run dev`
3. **訪問介面**：通常為 `http://localhost:5173`
